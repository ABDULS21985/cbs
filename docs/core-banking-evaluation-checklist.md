# Core Banking Capability Evaluation Checklist

> Use this checklist alongside the main capability reference to score vendors, assess gaps, or plan a modernisation roadmap.

## Scoring Guide

| Rating | Meaning |
|--------|---------|
| ✅ Full | Capability is production-ready, natively supported |
| 🟡 Partial | Available but requires customisation or has limitations |
| ❌ Gap | Not supported; requires third-party or custom build |
| 🔲 N/A | Not required for this institution's context |
